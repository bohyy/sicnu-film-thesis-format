# 参考文献与引文格式

川师 2026 版模板要求参考文献遵照 GB/T 7714-2015。正文引文可用顺序编码制或著者-出版年制，但全文必须统一。影传方向中文硕士论文若没有特别要求，通常优先保持或采用顺序编码制。

This user's thesis may use circled-number footnotes `①②③` as the visible citation/source-note system. When that convention is present, preserve it: the正文 circled marker points to the same-page bottom footnote, the footnote contains the source information, and numbering restarts from `①` on each page. The文末 `参考文献` list should still be formatted consistently with GB/T 7714 when present.

## 1. 顺序编码制

正文引用以文献出现顺序连续编号，置于方括号内：

- 单篇：`……相关研究认为互动叙事具有程序性特征[1]。`
- 多篇：`……已有研究从媒介融合和游戏叙事两个方向展开讨论[2-4]。`
- 同处多篇：`……相关问题已有论述[3,7,12]。`

参考文献表按正文引用顺序排列。

## 2. 著者-出版年制

正文引用以作者和出版年标注：

- `……互动叙事需要处理文本结构与用户路径之间的关系（Ryan, 2001）。`
- `Murray（1997）将数字媒介中的行动感视为重要审美经验。`

参考文献表按著者字顺和出版年排列。中文文献可按拼音或笔画；外文文献按字母顺序。

## 3. 常用顺序编码格式

### 学术期刊

`[序号] 作者. 文题[J]. 刊名, 年, 卷(期): 起止页码.`

Example:

`[1] 刘彻东. 中国的青年刊物: 个性特色为本[J]. 中国出版, 1998(5): 38-39.`

### 专著

`[序号] 作者. 书名[M]. 版本项. 出版地: 出版者, 出版年: 页码.`

Translated book:

`[序号] 作者. 书名[M]. 译者, 译. 版本项. 出版地: 出版者, 出版年: 页码.`

Example:

`[2] 中国社会科学院语言研究所词典编辑室. 现代汉语词典[M]. 修订本. 北京: 商务印书馆, 1996.`

### 学位论文

`[序号] 作者. 题名[D]. 授予单位所在地: 授予单位, 授予年份: 页码.`

Use for MA/PhD theses cited from CNKI or university repositories.

### 论文集/会议论文

`[序号] 作者. 文题[C]//论文集责任者. 论文集名. 出版地: 出版者, 出版年: 起止页码.`

Example:

`[3] 张忠智. 科技书刊的总编(主编)的角色要求[C]//中国科学技术期刊编辑学会建会十周年学术研讨会论文汇编. 北京: 中国科学技术期刊编辑学会学术委员会, 1997: 33-34.`

### 专利文献

`[序号] 专利申请者或所有者. 专利题名: 专利号[P]. 公告日期.`

Example:

`[4] 张凯军. 轨道火车及高速轨道火车紧急安全制动辅助装置: 201220158825.2[P]. 2013-03-27.`

### 技术标准

`[序号] 发布单位. 标准名称: 标准代号[S]. 出版地: 出版者, 出版年.`

### 电子资源

Use the appropriate type and carrier code such as `[M/OL]`, `[J/OL]`, `[D/OL]`, `[EB/OL]`, and include cited/access date and URL/DOI.

`[序号] 作者. 题名[文献类型标识/载体标识]. 出版地: 出版者, 出版年[引用日期]. 获取和访问路径或 DOI.`

Example:

`[5] 赵学功. 当代美国外交[M/OL]. 北京: 社会科学文献出版社, 2001[2014-06-11]. http://www.example.com.`

## 4. 文献类型标识

- 普通图书：`[M]`
- 期刊文章：`[J]`
- 学位论文：`[D]`
- 会议论文/论文集：`[C]`
- 报纸文章：`[N]`
- 标准：`[S]`
- 专利：`[P]`
- 电子公告/网页等：`[EB/OL]`
- 数据库：`[DB/OL]`

## 5. 影传方向处理建议

- 影视作品、互动影视、游戏、纪录片作为分析案例时，正文中使用书名号：`《Late Shift》`、`《黑镜：潘达斯奈基》`、`《隐形守护者》`。
- 若只是案例对象，通常不必强行列入参考文献；若引用了具体访谈、平台资料、官方页面、评论或数据，应按电子资源或网页资料著录。
- 影片年份、平台、导演、制作公司等信息可在正文首次出现时交代，或放入案例表格。
- 学术论文中的网络资料必须保留访问日期，避免只写裸 URL。
